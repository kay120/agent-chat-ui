# Git 工作流自动化

## 快速开始

```bash
# 查看帮助
./git-workflow.sh help

# 创建新分支
./git-workflow.sh new feature-name

# 快速提交推送
./git-workflow.sh quick "feat: 提交信息"

# 合并到 main
./git-workflow.sh merge
```
